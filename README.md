🎯 X-vs-AI

A classic Tic-Tac-Toe game where you play against an unbeatable AI. 

No random moves. No easy mode. The AI uses the "Minimax algorithm" — it looks ahead at every possible move, every possible response, all the way to the end of the game, and picks the move that guarantees it never loses.
You cannot beat it. The best you can do is draw.


🧠 How the AI works

AI uses Minimax Algorithm to Recursively evaluate all possible future board states
AI chooses the move with the highest possible score
With perfect play, the AI never loses


🎮 How to Play

1. If You are "X" (moves first)
2. AI is "O" (responds instantly)
3. Take turns placing your marks on a 3x3 grid
4. First to get "3 in a row" (horizontal, vertical, or diagonal) wins
5. If the grid fills up — it's a "draw"
